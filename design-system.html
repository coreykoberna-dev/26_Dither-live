05 / brand system

Brand

The brand system sits between the foundational tokens and the style library. It defines the two identity primitives that appear in product chrome: the animated pixel mark and the logo type.

01 primitive Animated pixel mark

Tokenized SVG character, theme-reactive at the pixel level, driven by walk and spell-cast animation states.

02 primitive Logo type

Compact GT Standard Mono wordmark with signal rails, a divider glyph, and the DTR/WZD support code.

shared rule Unboxed identity

The mark and wordmark sit directly in the interface. Do not wrap them in badges, chips, frames, or decorative logo cards.

brand subsection / animated pixel mark

Animated pixel mark

The Dither Wizard mage mark is not a raster logo and not a smooth-vector mascot. It is an SVG pixel sprite generated from approved directional mage anchors, rendered as individual rectangle cells, and recolored through the same dynamic roles as the product interface.

header sequence
Live header mark

Cycles through right, up, left, and down walk states, then plays orb and burst casts. Hover or focus cycles through elemental summon casts in the mage's current facing direction.

source anchors South, north, east, west PNGs

Approved directional art in assets/images locks the hood, robe volume, staff placement, face void, hand reads, and direction-specific silhouettes.

logical grid 72 by 84 cells

The encoder crops each anchor, fits it into the logical frame, quantizes the palette, and emits compact character rows.

renderer SVG rect pixels

dither-mage-vector.js converts every nontransparent cell to a rect, applies a mage-px-* class, and animates by offsetting pixels.

motion contract Shared clock, crisp cells

Idle, walk, and spell states use a single animation clock, cached vector-pixel frames, and low-amplitude bob and stride offsets so every instance stays crisp and synchronized.

brand subsection / logo type

Logo type

The final logo type is the Figma outlined Stack lockup: DITHER over WIZARD, built as two equal-width rows with tight leading, no divider, no tagline, and no support code inside the mark.

type constructionOutlined source

The source lockup began as GT Standard Mono in Figma and now ships as outlined vector geometry, preserving the exact row weight and edge profile.

geometry229 x 54 source

Both words share the same row width. The lockup uses the Figma outlined geometry directly so the corners and counters stay crisp.

lockup relationshipMark plus type

The mage pixel mark is the only character mark. The wordmark sits to its right without dividers, icons, subtitles, or secondary labels inside the logo type.

texture mappingTool-matched cutout

The logotype samples Dither Magic into a tiny canvas, mirrors the active tool stack, then renders with logo-only contrast, Bias, Epsilon Glow, and source-tone guards for legibility.

09 / behavior

State contract

Every interactive primitive should share one vocabulary for feedback. The styling can feel cinematic, but the behavior must remain familiar.

default

Neutral fill, clear outline, no decorative accent.

selectedselected

Primary role plus label or glyph, never color alone.

loadingscanning

Stable layout with scan, meter, or skeleton feedback.

warningunsaved

Warns without blocking; action remains available.

errorexport risk

Semantic color plus human-readable label.

10 / motion

Motion grammar

Motion should feel like calibrated instrumentation: sweep, scan, lock, compile, cast. Hover and focus respond immediately, loops stay subtle, and reduced motion disables nonessential movement.

sweepattention

Used for scanning diagnostics and dynamic-color proof only.

scanprocessing

Progress, loading, hover feedback, and export estimation can scan while layout stays fixed.

castbrand reaction

The mage mark reacts on hover, focus, click, and busy state.

11 / access

Accessibility

Contrast, keyboard focus, reduced motion, and color-independent status are product requirements, not polish passes.

contrast

Text and state labels must meet WCAG contrast in light, dark, and seeded schemes.

keyboard

Topbar, upload, presets, controls, timeline, export, footer, and system links remain reachable.

motion

Reduced motion disables sprite loops, scan sweeps, and decorative reveal movement.

status

Every warning, error, selected, or success state has a label, not only color.
